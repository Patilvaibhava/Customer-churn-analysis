# Customer-churn-analysis
Customer churn analysis using SQL &amp; Power BI

# Overview
This project analyzes customer churn using SQL and Power BI to identify key drivers of churn and recommend retention strategies.

# Tools Used
- MySQL
- Power BI

# Key Insights
- Low engagement customers have higher churn risk
- Payment delays strongly impact churn
- High-value customers need targeted retention
